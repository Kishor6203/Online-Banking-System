# 🏦 Online Banking System (Frontend)

## 📌 Overview
The **Online Banking System** is a responsive frontend web application built using **HTML, CSS, and JavaScript**.  
It simulates basic banking operations like account dashboard, balance view, transaction history, and fund transfer UI.

> ⚠️ This project is frontend-only and does not include backend or real banking integration. It is created for educational and UI/UX practice purposes.

---

## 🚀 Features
- 🧾 User Login Page (UI only)
- 🏠 Dashboard with account overview
- 💰 Balance display (static or simulated)
- 📊 Transaction history page
- 💸 Fund transfer interface (UI only)
- 📱 Fully responsive design
- 🎨 Clean and modern banking UI

---

## 🛠️ Technologies Used
- HTML5 – Structure
- CSS3 – Styling and responsive design
- JavaScript (ES6) – Interactivity and logic

---

## 💡 Future Improvements
- Add backend using spring boot  
- Implement real authentication system  
- Store transactions using database
